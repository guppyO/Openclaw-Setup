# Revenue OS Prompt

Build the strongest currently supportable autonomous revenue operating system on top of OpenClaw, Codex, and ChatGPT sign-in.

Done when:

- the repo can generate and maintain a real operating state,
- official-source runtime verification is recorded and refreshable,
- portfolio, treasury, skill-intake, update, and dashboard loops run locally,
- OpenClaw prod/stage/lab configs and deployment scripts are present,
- auth-dependent gaps are isolated with exact handoff notes instead of blocking the rest of the system.
